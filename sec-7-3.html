Section 7.3 Logarithms

Activity 7.3.1. Solving Equations.

(a)

Compare the two equations below. The first is a quadratic equation, and the second is an exponential equation. You can probably solve both of these equations in your head.

\begin{equation*} ~~~~\text{(1)}~~~x^2 = 25 ~~~~~~~~~~~~~~~~~~~~~~~\text{(2)}~~~2^x = 8 \end{equation*}

Solutions:

Let's analyze how we get the solutions. In the first equation, \(x\) is squared, so we perform the opposite operation to solve, or take the square root of both sides:

\begin{align*} \sqrt{x^2} \amp = \pm \sqrt{25}\\ x \amp = \pm 5 \end{align*}

In the second equation, \(x\) is an exponent. We ask ourselves "What power do we need on base 2 to get 8?" Finding a power is called taking a logarithm.

\begin{align*} \log_2 {2^x} \amp = \log_2{8}\\ x \amp = 3 \end{align*}

(b)

Now compare the two equations below. One is cubic and one is exponential:

\begin{equation*} ~~~~\text{(1)}~~~x^3 = 17 ~~~~~~~~~~~~~~~~~~~~~~~\text{(2)}~~~10^x = 135 \end{equation*}

These are not so easy to solve in your head. But we use the same techniques as before:

\begin{equation*} ~~~~\sqrt[3]{x^3} = \sqrt[3]{17} ~~~~~~~~~~~~~~~~~~~~~~~\log_{10}{10^x} = \log_{10}{135} \end{equation*}

But this time, we use a calculator to evaluate the answers. Locate the log button on your calculator.

The solution of the first equation answers the question "What number must I cube to get 17?" The solution of the second equation answers the question "To what power must I raise 10 to get 135?"

Activity 7.3.2. Definition of Logarithm.

We use logarithms to solve exponential equations.

\(\log_b {x}\) means "to what power must I raise \(b\) to get \(x\text{?}\)"

For example, \(~\log_4 {64} = \alert{3}~\) because \(~4^\alert{3} = 64\)

Activity 7.3.3. Logarithms and Inverse Operations.

We use inverse operations to solve equations. For example, cubing and taking cube roots are inverse operations. Each operation undoes the effect of the other. Start with any number, say 5, cube it, and then take the cube root. We get back to the original number:

\begin{equation*} 5 \rightarrow \text{cube} \rightarrow 5^3 = 125 \rightarrow \text{take cube root} \rightarrow \sqrt[3]{125} = 5 \end{equation*}

We can write the two operations in one step, like this: \(~~\sqrt[3]{5^3} = 5~~\text{.}\) The same thing works for any number \(x\text{,}\) so \(~~\sqrt[3]{x^3} = x~~\text{.}\)

(b)

Logarithms and exponentials are inverse operations. Taking a logarithm base \(b\) is the opposite of evaluating an exponential function with base \(b\text{.}\) For example, let \(b=3\text{,}\) and start by start by evaluating \(b^x\) with \(x=4\text{.}\) Then take the log base 3 of the result:

\begin{equation*} x=\alert{4} \rightarrow \text{exponential function} \rightarrow 3^4 = 81 \rightarrow \text{logarithm} \rightarrow \log_3{81} = \alert{4} \end{equation*}

Or, \(~\log_3 {3^4} = 4\text{.}\) So we have the rule: \(~\log_b {b^x} = x\)

Activity 7.3.5. Atmospheric Pressure.

The atmospheric pressure decreases with altitude above the surface of the earth, according to the formula

\begin{equation*} P(h) = 30(10^{-0.09h}) \end{equation*}

where \(h\) is altitude in miles and \(P\) is atmospheric pressure in inches of mercury.

(a)

Use technology to graph this function for \(0 \le h \le 10\text{.}\)

(b)
