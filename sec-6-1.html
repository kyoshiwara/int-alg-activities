Section 6.1 Integer Exponents

Activity 6.1.1. Power Functions.

(a)

State the values of \(k\) and \(p\) for the two examples below.

  1. \(V=f(r) = \frac{4}{3} \pi r^3,~~\) the volume of a sphere as a function of its radius

  2. \(L=f(T)= 0.8125 T^2,~~\) the length of a pendulum as a function of its period

(b)

Write each expression as a power function using negative exponents. State the values of \(k\) and \(p\text{.}\)

  1. \(\displaystyle h(s)=\dfrac{9}{s^3}\)

  2. \(\displaystyle f(v)=\dfrac{3}{8v^6}\)

  3. \(\displaystyle g(t)=\dfrac{1}{(5t)^4}\)

Activity 6.1.2. The Trebuchet.

In the middle ages in Europe, castles were used as defensive strongholds. An attacking force would build a huge catapult called a trebuchet to hurl rocks and scrap metal inside the castle walls. The engineers could adjust its range by varying the mass (or weight) of the missiles. The mass, \(m\text{,}\) of the projectile is inversely proportional to the square of the distance, \(d\text{,}\) to the target.

(a)

Use a negative exponent to write a formula for \(m\) as a function of \(d\text{,}\) \(m=f(d)\text{.}\)

(b)

The engineers test the trebuchet with a 30-pound lead ball, which lands 948 yards away. Find the constant of proportionality, and rewrite your formula for \(m\text{.}\) (Round your value of \(k\) to the nearest million.)

(c)

Graph \(m=f(d)\text{.}\)

(d)

The trebuchet is 500 yards from the courtyard within the castle. What size missile will hit the target?

(e)

The largest missile the trebuchet can handle is 300 pounds. If the attackers would like to hurl a 300-pound rock at the castle, how close must they bring their trebuchet?

Activity 6.1.3. Radar.

Airplanes use radar to detect the distances to other objects. A radar unit transmits a pulse of energy which bounces off a distant object, and the echo of the pulse returns to the sender. The power, \(P\text{,}\) of the returning echo is inversely proportional to the fourth power of the distance, \(d\text{,}\) to the object.

(a)

A radar operator receives an echo of \(5 \times 10^{-10}\) watt from an aircraft 2 nautical miles away. Express the power of the echo received in picowatts. (1 picowatt \(= 10^{-12}\) watt.)

(b)

Write a function that expresses \(P\) in terms of \(d\) using negative exponents. Use picowatts for the units of power.

(c)

Complete the table of values for the power of the echo received from objects at various distances.

\(d\) (nautical miles) 4 5 7 10
\(P\) (picowatts) \(\hphantom{0000000}\) \(\hphantom{0000000}\) \(\hphantom{0000000}\) \(\hphantom{0000000}\)

(d)

Sketch a graph of \(P\) as a function of \(d\text{.}\) Use units of picowatts on the vertical axis.

(e)

Radar units can typically detect signals as low as \(10^{-13}\) watt. How far away is an aircraft whose echo is \(10^{-13}\) watt? (Hint: Convert \(10^{-13}\) watt to picowatts.)

Activity 6.1.4. Using the Laws of Exponents.

(a)

Simplify if possible using the first law.

  1. \(\displaystyle x^6 \cdot x^{-3}\)

  2. \(\displaystyle x^{-6} \cdot x^3\)

  3. \(\displaystyle x^{-6} \cdot x^{-3}\)

  4. \(\displaystyle x^3 + x^{-6}\)

  5. \(\displaystyle x^{-3} - x^{-6}\)

  6. \(\displaystyle x^3 + x^{-3}\)

(b)

Simplify if possible using the second law.

  1. \(\displaystyle \dfrac{x^6}{x^3}\)

  2. \(\displaystyle \dfrac{x^{-6}}{x^3}\)

  3. \(\displaystyle \dfrac{x^6}{x^{-3}}\)

  4. \(\displaystyle \dfrac{x^{-6}}{x^{-3}}\)

  5. \(\displaystyle \dfrac{x^3}{x^{-6}}\)

  6. \(\displaystyle \dfrac{x^{-3}}{x^{-6}}\)

(c)

Simplify if possible using the third law.

  1. \(\displaystyle (x^6)^{-3}\)

  2. \(\displaystyle (x^{-6})^3\)

  3. \(\displaystyle (x^{-6})^{-3}\)

  4. \(\displaystyle (-x)^{-3}\)

  5. \(\displaystyle (-x)^{-6}\)

  6. \(\displaystyle (-x^3)^{-6}\)

(d)

Simplify if possible using the fourth and fifth laws.

  1. \(\displaystyle (2x)^{-3}\)

  2. \(\displaystyle (\dfrac{2}{x})^{-3}\)

  3. \(\displaystyle (2x^4)^{-3}\)

  4. \(\displaystyle (2x^{-4})^3\)

  5. \(\displaystyle (2x^{-4})^{-3}\)

  6. \(\displaystyle (-2x^{-4})^{-3}\)

  7. \(\displaystyle \left(\dfrac{2}{x^{-4}}\right)^3\)

  8. \(\displaystyle \left(\dfrac{2}{x^{-4}}\right)^{-3}\)

(e)

Which of the following statements are true for all non-zero values of \(a\) and \(b\text{?}\) (You may want to try out some specific values of \(a\) and \(b\) before you decide!)

  1. \(\dfrac{1}{a} + \dfrac{1}{b} = \dfrac{1}{a+b}~\text{?}\)

  2. \(a^{-1} + b^{-1} = (a+b)^{-1}~\text{?}\)

  3. \(a^2 + b^2 = (a+b)^2~\text{?}\)

  4. \(a^{-2} + b^{-2} = (a+b)^{-2}~\text{?}\)

  5. \(\dfrac{1}{a^{-1}} + \dfrac{1}{b^{-1}} = a + b~\text{?}\)

  6. \(\dfrac{1}{a^{-1} + b^{-1}} = a + b~\text{?}\)

Subsection 6.1.1 Check Your Understanding

  1. Explain the difference between each pair of expressions.

    1. \(-2^3\) and \(2^{-3}\)

    2. \(-x^4\) and \(x^{-4}\)

    3. \(-2^n\) and \(2^{-n}\)

  2. Write a power function for "\(y\) varies inversely with the cube of \(x\text{.}\)"

  3. How do you convert a value in watts to picowatts?

  4. Explain how to decide which law of exponents to use when simplifying an expression.

Subsection 6.1.2 Wrap-Up

In this Lesson, we worked on the following skills and goals related to powers and roots:

  • Evaluate negative exponents

  • Write a formula for a power function

  • Graph and analyze power functions

  • Solve equations involving negative exponents

  • Simplify expressions using the laws of exponents
