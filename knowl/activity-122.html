Activity 8.2.2. Equivalent Fractions.

Which expressions are equivalent to the given fraction?

(a)

\(\dfrac{3}{5}\)

  1. \(\displaystyle \dfrac{3x}{5x}\)

  2. \(\displaystyle \dfrac{3-x}{5-x}\)

  3. \(\displaystyle \dfrac{30+x}{x+50}\)

  4. \(\displaystyle \dfrac{30x}{50x}\)

(b)

\(\dfrac{v-1}{v+1}\)

  1. \(\displaystyle -1\)

  2. \(\displaystyle \dfrac{1-v}{1+v}\)

  3. \(\displaystyle -\dfrac{1-v}{v+1}\)

  4. \(\displaystyle \dfrac{v+1}{v-1}\)

(c)

\(\dfrac{2w-6}{w}\)

  1. \(\displaystyle -4\)

  2. \(\displaystyle \dfrac{6-2w}{-w}\)

  3. \(\displaystyle 2-\dfrac{6}{w}\)

  4. \(\displaystyle -\dfrac{2w+6}{w}\)

(d)

\(\dfrac{3a+2}{3a-2}\)

  1. \(\displaystyle \dfrac{3a-2}{3a+2}\)

  2. \(\displaystyle \dfrac{3a-2}{2-3a}\)

  3. \(\displaystyle -1\)

  4. \(\displaystyle \dfrac{2a+3}{2a-3}\)
