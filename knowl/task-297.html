Finally, we'll use algebra to sove the equation. Recall the definition of absolute value:

\begin{align*} \abs{x} = \begin{cases} x \amp \text{if } x\ge 0\\ -x \amp \text{if } x\lt 0 \end{cases} \end{align*}

Now substitute \(x-3\) for \(x\text{:}\)

\begin{align*} \abs{x-3} = \begin{cases} x-3 \amp \text{if } x-3\ge 0\\ -(x-3) \amp \text{if } x-3\lt 0 \end{cases} \end{align*}
