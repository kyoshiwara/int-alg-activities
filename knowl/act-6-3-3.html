Activity 6.3.3. Laws of Exponents.

(a)

Simplify by using the laws of exponents. Write your answers using radicals.

  1. \(\displaystyle x^{3/4}x^{5/4}\)

  2. \(\displaystyle \left(x^{3/2}\right)^{5/2}\)

  3. \(\displaystyle \dfrac{z^{1/3}}{z^{2/3}}\)

  4. \(\displaystyle \dfrac{w^{1/5}w^{-3/5}}{w^{-4/5}}\)

(b)

Simplify by using the laws of exponents. Write your answers using radicals.

  1. \(\displaystyle b^{1/3}b^{2/5}\)

  2. \(\displaystyle \dfrac{a^{5/4}}{a^{-4/3}}\)

  3. \(\displaystyle \left(-3c^{4/3}\right)^{-2}\)

  4. \(\displaystyle \dfrac{\left(a^{3/2}b^{-1/4}\right)^{1/3}}{a^{3/8}}\)
