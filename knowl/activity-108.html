Activity 7.3.4. Solving Base 10 Equations.

We can now solve exponential equations that have base 10.

\begin{align*} 10^x \amp = 150~~~~~~~~~~~~\text{Take the log base 10 of both sides.}\\ \log {10^x} \amp = \log {150}~~~~~~~~~~~~\text{Simplify each side.}\\ x \amp = 2.1761~~~~~~~~~~~~\text{Check:}~10^{2.1761} = 150.003 \end{align*}
