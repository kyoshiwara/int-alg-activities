Activity 8.4.3. Polynomial Division.

If an algebraic fraction cannot be reduced, we can simplify it by dividing the denominator into the numerator.

(a)

Monomial divisor. Divide the monomial into each term of the numerator.

\(~~~\dfrac{2m^6-15m^4+7}{-5m^3}\)

(b)

Binomial divisor. We follow the procedure for long division. First, prepare the fraction:

Write the fraction \(\dfrac{\text{numerator}}{\text{denominator}}\) this way: denominator )\overline{numerator}

So the denominator becomes the divisor. Write both polynomials in descending powers of the variable, and insert terms with coefficient 0 if any powers are missing.

(c)

Divide \(~~~~\dfrac{2x^3 - 3x^2-2x+4}{x+1}\)

(d)

Divide \(~~~~\dfrac{7-3t^3-23t^2+10t^4}{2t+3}\)
