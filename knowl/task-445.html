Caution! Be careful when using the properties of logarithms. Which expressions in each group are equivalent (if any)?

  1. \(\displaystyle \log \dfrac{x}{25}~~~~~~~~\dfrac{\log x}{\log 25}~~~~~~~~\log x - \log 25~~~~~~~~\log (x-25)\)

  2. \(\displaystyle \log (25x)~~~~~~~~\log 25 + \log x~~~~~~~~\log (x+25)~~~~~~~~\log x \log 25\)

  3. \(\displaystyle \log (x^2)~~~~~~~~(\log x)^2~~~~~~~~2 \log x~~~~~~~~\log 2x\)
