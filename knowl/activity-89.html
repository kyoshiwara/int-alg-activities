Activity 6.4.2. Operations on Radicals.

We can combine the radicands in products or quotients of radicals as long as both radicals have the same index, but we cannot combine radicands in sums or differences. You can check that:

\begin{align*} \sqrt{12}~\sqrt{3} = \sqrt{36}~~~~~~~~\amp \text{and}~~~~~~~~ \dfrac{\sqrt{12}}{\sqrt{3}} = \sqrt{4}\\ \text{but}~~\sqrt{12}+\sqrt{3} \not= \sqrt{15}~~~~~~~~\amp \text{and}~~~~~~~~ \sqrt{12}-\sqrt{3} \not= \sqrt{9} \end{align*}

(a)

Simplify the following products and quotients.

  1. \(\displaystyle \sqrt[3]{4x^2}~\sqrt[3]{16x}\)

  2. \(\displaystyle \sqrt[4]{2m^3}~\sqrt[4]{8m}\)

  3. \(\displaystyle \dfrac{\sqrt{18xy^3}}{\sqrt{2y}}\)

  4. \(\displaystyle \dfrac{\sqrt[3]{200a^8}}{\sqrt[3]{5a^2}}\)

(b)

True or false?

  1. \(\sqrt{7} + \sqrt{7} = \sqrt{14}~\text{?}\)

  2. \(\sqrt{7} + \sqrt{7} = 2\sqrt{7}~\text{?}\)

  3. \(\sqrt{7} - \sqrt{5} = \sqrt{2}~\text{?}\)

  4. \(6\sqrt{5} - 4\sqrt{5} = 2\sqrt{5}~\text{?}\)

(c)

Simplify and combine like terms.

  1. \(\displaystyle 5\sqrt{6} - 2\sqrt{6} + 6\sqrt{2}~\)

  2. \(\displaystyle 8\sqrt[3]{a^2} - 5\sqrt[3]{a} - 5\sqrt[3]{a^2}~\)

  3. \(\displaystyle 2\sqrt{75} + 3\sqrt{48} \)

  4. \(\displaystyle \sqrt[3]{81} + \sqrt[3]{24} - 3\sqrt[3]{3}~\)

(d)

Simplify the products. Use the distributive law when appropriate.

  1. \(\displaystyle (3\sqrt{2})(2\sqrt{5})\)

  2. \(\displaystyle (3\sqrt{5})(2\sqrt{5})\)

  3. \(\displaystyle (2\sqrt{15})(3\sqrt{6}) \)

  4. \(\displaystyle 5(2\sqrt{3} - \sqrt{5})\)

  5. \(\displaystyle 4\sqrt{2}(4\sqrt{3} + 2\sqrt{6})\)

  6. \(\displaystyle (3\sqrt{x} - \sqrt{5})(2\sqrt{x} + 3\sqrt{5})\)
