Activity 10.3.2. Changing the Base.

It is useful to express any exponential function with base \(e\text{.}\) This is easy to do.

(a)

Which law of exponents tells us that \(~(e^k)^t = e^{kt}~\text{?}\)

(b)

Follow the steps to write \(P = 25(\alert{10})^t\) as \(P = 25 (\alert{e^k})^t\)

We need to replace \(10\) by \(e^k\text{.}\) Solve the equation \(~10 = e^k~\) for \(k\text{.}\) Round your answer to tenths.

Substitute your value for \(k\) into the formula for \(P\text{.}\)

Check that the two formulas for \(P\) give the same outputs for \(x = 1,~x=2,~\) and \(x=3\text{.}\)

(c)

Rewrite each exponential function with base \(e\text{:}\)

  1. \(\displaystyle H=12(1.05)^t\)

  2. \(\displaystyle N=80(0.16)^t\)
