Activity 4.1.3. Summary: Solving Quadratic Equations.

(a)

List four algebraic methods for solving quadratic equations.

(b)

Which method(s) work on any quadratic equation?

(c)

Which method is fastest if one side of the equation is a perfect square?

(d)

For each equation, name the easiest method for solving, then solve.

  1. \(\displaystyle x^2-7=0\)

  2. \(\displaystyle x^2=7x\)

  3. \(\displaystyle x^2-7x=8\)

  4. \(\displaystyle (x-7)^2=8\)

  5. \(\displaystyle x^2=7x-8\)

  6. \(\displaystyle x^2-8x=4\)
