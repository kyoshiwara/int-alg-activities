Task 4.3.2.b.

How can we find the equation of a parabola?

  • If we know the \(x\)-intercepts of the parabola, then we use the factored form.

  • If we know the vertex, we use the vertex form.

  • In each case, we must know one other point on the parabola in order to find the value of \(a\text{.}\)

  1. Write an equation for a parabola that has \(x\)-intercepts at \((2,0)\) and \((-3,0)\text{.}\) (There are many possible answers.)

  2. Write an equation for another parabola that has the same \(x\)-intercepts.

  3. Write an equation for the parabola that has \(x\)-intercepts at \((2,0)\) and \((-3,0)\) and \(y\)-intercept at \((0,3)\text{.}\)
