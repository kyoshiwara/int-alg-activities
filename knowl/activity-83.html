Activity 6.1.4. Using the Laws of Exponents.

(a)

Simplify if possible using the first law.

  1. \(\displaystyle x^6 \cdot x^{-3}\)

  2. \(\displaystyle x^{-6} \cdot x^3\)

  3. \(\displaystyle x^{-6} \cdot x^{-3}\)

  4. \(\displaystyle x^3 + x^{-6}\)

  5. \(\displaystyle x^{-3} - x^{-6}\)

  6. \(\displaystyle x^3 + x^{-3}\)

(b)

Simplify if possible using the second law.

  1. \(\displaystyle \dfrac{x^6}{x^3}\)

  2. \(\displaystyle \dfrac{x^{-6}}{x^3}\)

  3. \(\displaystyle \dfrac{x^6}{x^{-3}}\)

  4. \(\displaystyle \dfrac{x^{-6}}{x^{-3}}\)

  5. \(\displaystyle \dfrac{x^3}{x^{-6}}\)

  6. \(\displaystyle \dfrac{x^{-3}}{x^{-6}}\)

(c)

Simplify if possible using the third law.

  1. \(\displaystyle (x^6)^{-3}\)

  2. \(\displaystyle (x^{-6})^3\)

  3. \(\displaystyle (x^{-6})^{-3}\)

  4. \(\displaystyle (-x)^{-3}\)

  5. \(\displaystyle (-x)^{-6}\)

  6. \(\displaystyle (-x^3)^{-6}\)

(d)

Simplify if possible using the fourth and fifth laws.

  1. \(\displaystyle (2x)^{-3}\)

  2. \(\displaystyle (\dfrac{2}{x})^{-3}\)

  3. \(\displaystyle (2x^4)^{-3}\)

  4. \(\displaystyle (2x^{-4})^3\)

  5. \(\displaystyle (2x^{-4})^{-3}\)

  6. \(\displaystyle (-2x^{-4})^{-3}\)

  7. \(\displaystyle \left(\dfrac{2}{x^{-4}}\right)^3\)

  8. \(\displaystyle \left(\dfrac{2}{x^{-4}}\right)^{-3}\)

(e)

Which of the following statements are true for all non-zero values of \(a\) and \(b\text{?}\) (You may want to try out some specific values of \(a\) and \(b\) before you decide!)

  1. \(\dfrac{1}{a} + \dfrac{1}{b} = \dfrac{1}{a+b}~\text{?}\)

  2. \(a^{-1} + b^{-1} = (a+b)^{-1}~\text{?}\)

  3. \(a^2 + b^2 = (a+b)^2~\text{?}\)

  4. \(a^{-2} + b^{-2} = (a+b)^{-2}~\text{?}\)

  5. \(\dfrac{1}{a^{-1}} + \dfrac{1}{b^{-1}} = a + b~\text{?}\)

  6. \(\dfrac{1}{a^{-1} + b^{-1}} = a + b~\text{?}\)
